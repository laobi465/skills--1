# 现代简约风格网页应用 UI 设计规则

设计一套现代简约风格的网页应用UI，整体视觉干净、专业、易用。

## ===== 严格禁用项 =====

1. 绝对禁止使用任何emoji、表情符号、图标化装饰元素
2. 禁止夸张设计：避免大渐变、强对比、炫光、霓虹、3D凸起等夸张视觉
3. 禁止毛玻璃效果：不使用backdrop-filter、玻璃拟态、半透明模糊背景
4. 禁止暗黑风格：不使用黑色、深灰、深紫色为主的深色主题，整体保持明亮
5. 禁止过度装饰：去掉不必要的装饰线条、背景纹理、花哨图案

## ===== 设计风格要求 =====

### 【色彩系统】
- 背景：纯白或极浅灰色作为主背景，干净明亮
- 主色：选择低饱和度的专业色系（如藏蓝、湖蓝、青绿、墨绿等），用于重点按钮、关键文字
- 辅助色：1-2种柔和的辅助色，用于标签、状态提示，避免鲜艳刺眼
- 文字：深灰为主，层级通过灰度和字重区分，不使用纯黑
- 分割线与边框：极浅灰色，若有若无

### 【组件规范】
- 按钮：圆角4-8px，扁平或轻微阴影，hover状态简洁
- 卡片：白色背景，细边框或极淡阴影，圆角适中
- 输入框：细边框，聚焦时主色高亮，简洁无多余装饰
- 导航：顶部或侧边，底色浅，选中态清晰不突兀

### 【布局与排版】
- 留白充足，内容区域呼吸感强
- 信息层级分明，重要内容突出但不夸张
- 字体使用现代无衬线体系，字重梯度清晰
- 整体对齐工整，视觉节奏稳定

### 【动效与交互】
- 过渡动画轻柔快速，不拖沓
- 状态变化反馈明确但不夸张
- 无炫技式动画效果
