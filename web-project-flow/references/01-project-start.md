# 项目起步提示词

## 完整版

请帮我开发一个网站项目，需求如下：

【网站类型】
例如：企业官网 / 个人博客 / 电商商城 / 后台管理系统 / 工具类网站 / 作品集展示站

【核心功能】
1. xxx
2. xxx
3. xxx

【技术栈要求】
- 前端：Vue3 / React / 原生HTML+CSS+JS
- 后端：Node.js / Python(Django/Flask) / PHP / 纯静态无需后端
- 数据库：MySQL / MongoDB / 不需要
- 其他：是否需要响应式、是否要深色模式、是否要登录系统

【设计风格】
- 配色：例如莫兰迪色系 / 科技蓝 / 简约黑白
- 风格：玻璃拟态 / 极简风 / 商务风 / 赛博朋克
- 参考：可以描述类似哪个网站的感觉

【页面清单】
- 首页
- 关于页
- 产品/服务页
- 联系页
- ……

【其他要求】
- 代码要模块化、有注释
- 适配手机端
- 有基础的动画过渡效果

请先确认以上需求是否完整，有需要我补充的地方请主动提问，然后再开始写代码。

## 精简版

帮我做一个 [网站类型，比如"个人博客"] 网站。
我不太清楚具体细节，请你先根据这个类型，列出一份标准的功能和页面清单，
然后主动问我哪些需要、哪些不需要，确认完再开始写代码。
技术栈默认用你最推荐的组合就行。
